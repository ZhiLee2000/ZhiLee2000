Color Models
How does RGB color model work?
The RGB color model stands for red, green, and blue. It is an additive color model, which means different amounts of red, green, and blue light are mixed to produce all other colors.
What are RGB and CMYK color models?
The CMYK color model is a subtractive color model and stands for cyan, magenta, yellow, and black. The RGB color model is an additive color model and stands for red, green, and blue.
How is the CMYK color model usually used?
The CMYK color model stands for cyan, magenta, yellow, and black. The CMYK color model is a subtractive color model used for mixing pigments.
